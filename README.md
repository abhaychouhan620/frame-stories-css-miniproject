# frame-stories-css-miniproject
A photography-themed mini project built using HTML and CSS.

## Features

* Sidebar navigation menu
* Photography gallery
* CSS hover effects
* Smooth transitions and animations
* Glassmorphism-inspired design
* Responsive layout concepts

## Technologies Used

* HTML5
* CSS3
* Font Awesome
* Google Fonts

## Project Overview

This project was created as part of my frontend development learning journey. It helped me practice CSS concepts such as Flexbox, transitions, animations, hover effects, and UI design. I also customized the project by adding a photography gallery, animations, and the "Frame Stories" theme.
